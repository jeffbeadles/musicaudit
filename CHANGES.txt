   Change for issue #4, to be able to disable rules
   Rule selection:
        --rule on the command line overrides all configuration.
        Otherwise, all rules are selected by default.
        enabled_rules limits the selection to the listed rules.
        disabled_rules removes listed rules from the selection.
        The resulting rule set is evaluated

    enabled_rules is an allow list.
    disabled_rules is a deny list.
